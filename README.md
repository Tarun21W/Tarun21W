<br/>

An OpenAI-compatible proxy that cuts input-token spend for coding agents through code-aware context optimization and opt-in, quality-based model routing — works with any IDE that accepts a base URL.

| Attribute | Detail |
| :--- | :--- |
| **Stack** | Python · FastAPI · httpx · tiktoken |
| **Scale** | Drop-in proxy in front of any OpenAI-compatible upstream (OpenRouter, OpenAI, LiteLLM) |
| **Performance** | Request-side dedup, stale tool-output collapse, history windowing |
| **Security** | Conservative, lossless-by-default transforms; never prunes tokens inside code |
| **Impact** | Reduces input tokens and routes by difficulty without changing the developer's tooling |
| **Repository** | [github.com/Tarun21W/ctxcut](https://github.com/Tarun21W) |

Open-source by design: composes mature primitives (routing, dispatch) and differentiates on the agentic-coding-specific structural cleanup.

</details>
